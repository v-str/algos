# Скользящее окно (Sliding Window)

## Когда использовать:
- Массив/строка
- Нужен непрерывный подмассив/подстрока
- "Максимальная/минимальная сумма длины k"
- "Без повторений"
- "Содержит все символы из ..."

## Триггерные слова:
подмассив, подстрока, непрерывный, длина k, максимум, минимум, уникальные символы

## Шаблон (C++):
// Расширяем окно правым указателем
// Сужаем окно левым, когда условие нарушается
// Обновляем ответ
